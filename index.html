One-time setup needed
Microphone & app install require a secure HTTPS link. Setup is free and takes ~2 minutes.
⚠️ Why mic doesn't work from a file
Browsers block microphone on file:// URLs by design — this is a security rule that cannot be bypassed. The app must be served over https:// to access the mic.
✅ Host free on GitHub Pages (2 minutes)
1
Create a free GitHub account
Go to github.com and sign up — free, no card needed.
→ github.com/signup
2
Create a new repository
Tap + → New repository → name it expert-ai → set Public → tick "Add README" → Create.
3
Upload and rename this file
Inside the repo tap "Add file" → "Upload files" → upload ExpertAI.html. Then click it → pencil icon → rename to index.html → Commit changes.
4
Enable GitHub Pages
Repo Settings → Pages → Source: "Deploy from branch" → Branch: main → / (root) → Save. Wait 60 seconds.
5
Open your HTTPS link on Android
Your app will be live at:
https://YOUR-USERNAME.github.io/expert-ai/
Open in Chrome on your phone — mic and PWA install will both work.
6
Install as Android app
In Chrome tap ⋮ → "Add to Home screen" → Install. Opens fullscreen with full mic access.
